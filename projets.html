PA2L : App de Matchmaking

PA2L est une application mobile conçue pour connecter les joueurs de padel avec des partenaires de même niveau. Elle facilite la réservation de terrains dans les clubs à proximité et propose un système de classement engageant inspiré des jeux vidéo compétitifs.

Carte Interactive

Affichage des clubs de padel à proximité avec indicateurs de distance, notes et créneaux disponibles.

Matchmaking Intelligent

Interface type "swipe" pour trouver des partenaires compatibles selon le niveau, avec chat intégré.

Système de Classement (Ranked)

Inspiré de Call of Duty : Progression du Bronze à la Légende avec effets visuels et statistiques détaillées.

Paiements Intégrés

Intégration fluide de Stripe et Apple Pay avec synchronisation automatique du calendrier.

Design Sportif Moderne

Interface audacieuse utilisant le Bleu Roi (#0052CC) et le Jaune Padel (#F7D229).

Application de Gestion des Ligues

Évolution d'une application Java en ligne de commande (CLI). Passage d'une version mono-utilisateur statique à une version multi-utilisateurs avec base de données et architecture structurée.

Contexte

Le projet initial consistait en un squelette d'application Java (code fourni). La logique de base était présente, mais manquait de persistance des données et de contrôles métier stricts.

Mes Réalisations

  • Gestion Temporelle : Ajout des dates d'arrivée et de départ.
  • Robustesse : Mise en place d'exceptions personnalisées.
  • Interface (CLI) : Modification des menus pour les admins.
  • Qualité logicielle : Implémentation de tests unitaires (JUnit).
  • Conception : Mise à jour du MCD et arbre heuristique.

Architecture 3-Tiers

  • 1. Présentation : Interface CLI
  • 2. Métier : Logique Java
  • 3. Données : Base de données

Habilitations

  • Admin Ligue : Écriture partielle
  • Super-Admin : Accès total
